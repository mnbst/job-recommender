プライバシーポリシー

最終更新日: 2025年1月

1. 収集する情報

本サービスでは、GitHub Appの認可を通じて以下の情報を取得します。

GitHub アカウント情報

  • ユーザーID・ログイン名
  • 表示名・メールアドレス（公開設定時のみ）
  • アバター画像URL
  • 公開リポジトリ情報（名前、説明、使用言語、README等）

2. データの保存

取得したデータは Google Cloud Firestore に保存されます。

  • セッション情報: 7日間
  • プロファイル・キャッシュ: 30日間

3. 第三者サービスへのデータ送信

Google Vertex AI (Gemini)

プロファイル生成のため、リポジトリ情報を送信します。

Perplexity AI

求人検索のため、生成されたプロファイルを送信します。

4. データの削除

ログアウト時にセッション・プロファイル・キャッシュは即座に削除されます。

5. Cookieの使用

セッション管理のためにCookieを使用しています（7日間有効）。

6. お問い合わせ

GitHubリポジトリのIssueにてお問い合わせください。
